# AtomicTree Engine Extension

Visualize and benchmark your AtomicTree NVM storage performance in real-time within Visual Studio Code.

## Features

- **🚀 Live Performance Metrics**: Monitor throughput, latency, and operation counts as they happen.
- **🗺️ Memory Heatmap**: Visualize NVM memory allocation and fragmentation patterns.
- **🛡️ Crash Recovery**: Monitor the status of persistent memory consistency and recovery logs.
- **♻️ GC Statistics**: Keep track of background garbage collection activity and its impact on performance.

## How to Use

1. **Install** this extension from the VS Code Marketplace.
2. **Setup** your C++ project using the AtomicTree library (see [documentation](https://github.com/ayeshaaa0134/atomic-engine)).
3. **Run** your code. The extension will automatically detect AtomicTree activity and display the visualizer.
4. Open the **AtomicTree Viz** activity bar icon to see your data!
